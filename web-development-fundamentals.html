In this course, you will learn the basics of developing a website from scratch using HyperText Markup Language (HTML), Cascading Style Sheets (CSS) and Javascript, all the way to live your own website.

After completing this course, you will able to:
  • Understand the Concept of World Wide Web
  • Structuring Web Page Content using HTML
  • Styling the Content with CSS
  • Create Dynamic Web Pages with Javascript
  • Publish Website to Live
