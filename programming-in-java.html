This course offers a solid foundation in Java Programming for students that without any programming experience. The syllabus starts with a basic introduction to Java and students will learn how to write their first simple Java application. Besides learning basic Java syntax and structure, object-oriented principles will also covered in this course.

After completing this course, you will able to:
  • Build basic Java Applications
  • Use Intellij IDEA
  • Compile and Run Java Applications
  • Declare and Use Variables
  • Use Different Types of Data Types
  • Use Operators and Expressions in Java
  • Implement Control Flow
  • Understand Object-Oriented Programming
  • Use Classes, Objects and Call Methods
  • Handle Exceptions
  • Test and Debug Java Applications
